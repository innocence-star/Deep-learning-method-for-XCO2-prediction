This project uses a CNN-Transformer hybrid deep learning model for high-precision XCO₂ prediction from satellite and remote sensing data.
Project Structure:
train.py: Implements the full training pipeline, including the CNN-Transformer model architecture, hyperparameter settings, training, validation, and model checkpoint saving.
utils.py: Contains inference code to load the trained model, execute predictions, and output the final XCO₂ prediction results.
Model results are now available in this repository. More data processing and training codes will be uploaded gradually.
