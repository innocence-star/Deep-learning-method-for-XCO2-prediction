This project uses a CNN-Transformer hybrid deep learning model for high-precision XCO₂ prediction from satellite and remote sensing data.

Model results have been uploaded. Additional processing codes are being updated.

## Project Structure
- train.py: Contains the complete model training pipeline, including the main CNN-Transformer architecture, hyperparameter settings, and full training process.
- utils.py: Used for model inference to load the trained model and output complete prediction results.
- Model Result: Estimated XCO2 data.
- README.md: Project description and usage instructions.
